Author: David Chandra Purnama

Shell is a responsive parent theme developed as a starting point for blogs. Its primary goal is to get your blog up and running without a lot of hassle and allow you to make quick customizations. Shell is based on Prototype theme. Shell allow developer to crete skin plugin or child theme with multiple skin.

This theme is a parent theme. What this means is that to customize it, you should be creating a child theme. Is it a theme framework? Some people might call it that. All you need to know is that it's a solid, yet flexible, starting point for any blog.

Features (back to top)

Shell is built from the rock-solid Hybrid Core theme framework, so it provides a great starting point with many useful features.

  • Per-post layouts that allow you to quickly change the layout of your site.
  • A smart breadcrumb menu that handles tons of permalink setups.
  • Built-in pagination for your home, archive, and search pages.
  • Post thumbnails that work with WordPress' featured image functionality and provide extra features.
  • Multiple sidebars that you can customize however you like.
  • Dynamic global, post, and comment CSS classes.
  • Uses the built-in WordPress menu system. No need to hack your navigation into place.
  • Context-aware action and filter hooks, so you don't have to learn WordPress conditional tags.
  • Some of the most-used filter hooks are shortcode-ready and have accompanying shortcodes.
  • Theme settings page that allows you to add custom meta boxes and settings.
  • A redefined template hierarchy that makes larger sites easier to design for.
  • Ability to create custom templates for any post type in multiple ways.
  • Easily add post-specific settings and not make users learn how to use custom fields.
  • Highly-configurable widgets that give you full control over their display.
  • Widget areas are content-aware, meaning that they only appear if they contain active widgets.
  • Skin Option.
  • And much, much more!

Child Theme

Since shell is a parent theme, you'll want to create a child theme if you plan on making any customizations. Don't know how to make a child theme? It's relatively simple. Just follow the below steps.

  • Create a theme folder in your /wp-content/themes directory called shell-child.
  • Then, create a style.css file within your theme folder.
  • At the top of your style.css file, add the below information.
/**
 * Theme Name: Shell Child
 * Theme URI: http://link-to-your-site.com
 * Description: Describe what your theme should be like.
 * Version: 0.1.0
 * Author: Your Name
 * Author URI: http://link-to-your-site.com
 * Tags: Add, Whatever, Tags, You, Want
 * Template: shell
 */

You don't need to import parent theme style

Note that child themes are now the officially-supported way of modifying themes in WordPress.
